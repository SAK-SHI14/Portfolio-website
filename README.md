## 🎬 Overview

A **premium, cinematic personal developer portfolio** built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies. Designed to make a powerful first impression with a dark theme, glassmorphism cards, fluid animations, and an animated neural-network particle canvas.

Deployed live at: **[playful-semolina-9c09f1.netlify.app](https://playful-semolina-9c09f1.netlify.app/)**

---

## ✨ Features

### 🎨 Design & UI
- **Dark cinematic theme** — Deep navy `#0D1117` with lavender, rose pink, and gold accents
- **Glassmorphism cards** — `backdrop-blur`, semi-transparent borders, frosted glass
- **Animated neural-network canvas** — WebGL-style particle system with mouse interaction
- **Floating gradient blobs** — Purple, pink, and gold ambient glow
- **3D tilt effect** — Perspective CSS transforms on project cards
- **Gradient scroll progress bar** — Top of viewport

### ⚡ Interactions & Animations
- **Typewriter effect** — Cycles through 5 roles with realistic typing + deletion
- **Cursor glow trail** — Smooth radial gradient following mouse
- **Scroll-reveal animations** — IntersectionObserver staggered fade-in
- **Animated count-up counters** — EaseOut cubic for stats
- **Glowing skill progress bars** — With shimmer animation
- **Parallax scrolling** — Blob drift on scroll

### 🛠 Functionality
- **Contact form → Gmail** — AJAX submission via FormSubmit.co, no backend needed
- **Dark / Light mode toggle** — Persisted via `localStorage`
- **Sticky glassmorphism navbar** — Active section highlighting on scroll
- **Responsive mobile-first** — Hamburger menu, adaptive grid layouts
- **Back-to-top button** — Floating gradient pill
- **Konami Code easter egg** 🎮 — `↑↑↓↓←→←→BA` triggers confetti
- **Confetti burst** — Canvas particle effect on CTA interactions
